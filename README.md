# 魂核（Soul Core）· 认知架构实验

> 探索「机器如何拥有类人意识」的个人研究项目。  
> **4.0 与 6.1.8** 是魂核的前两个时代：**规则与认知架构**；它们**不是**神经形态仿真。

本仓库为**持续维护的公开存档**，欢迎学习、拆分与二次开发——不是废弃代码。

---

## 版本说明

本仓库包含两个**历史版本**（**不再在此仓库内更新**）：

| 文件 | 核心路线 | 说明 |
|------|----------|------|
| **`魂核4.0.txt`** | **规则系统**：人格、情感、记忆、决策、关系等（含 `soul_public_v4/` 文件清单与多段代码汇编） | 单文件**汇编稿**（含树状清单与片段），**非**可一键运行的单个 `.py` |
| **`魂核6.1.8.py`** | **认知架构**：动机、自指涉、叙事自我、自由能等 | **完整 Python 单文件**，可直接供 `import` 或按需拆分 |

**重要**：上述版本**不包含「脉冲神经元 / SNN」**，属于**符号主义与认知架构**路线。

---

## 扩展名为什么不同（`.txt` vs `.py`）

- **`魂核6.1.8.py`**：正文为**合法 Python 模块结构**，用 `.py` 便于高亮、导入与社区习惯。  
- **`魂核4.0.txt`**：内容为**目录清单 + 多段代码合订**，整文件**不能**当作一个无改动的 `.py` 直接通过解释器执行；若强行改扩展名会误导读者。因此保留 **`.txt`**，表示「公开汇编稿」。

若你希望 4.0 也变成标准包结构，需要自行拆成 `soul_public_v4/` 多文件（超出本仓库当前范围）。

---

## 与 v11.0 的关系（边界）

**v11.0** 为另一条路线：**神经形态计算**（大规模脉冲网络、调质、记忆与情绪动力学等）。**v11.0 完整实现不在本仓库**，以免与「规则时代」稿件混淆；若有单独公开计划，会另作说明。

---

## 使用提示

- **6.1.8**：可作库导入；若直接 `python 魂核6.1.8.py`，视文件内是否含 `if __name__ == "__main__"` 而定。依赖以文件内 `import` 为准。  
- **4.0**：请按文中结构**自行拆模块**或复制片段到项目中使用。
